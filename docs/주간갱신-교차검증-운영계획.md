# 주간 상품 갱신·교차검증·콘텐츠 운영계획

작성일: 2026-07-22
운영 방식: 수동 큐레이션 1단계, 자동화 2단계

## 1. 매주 운영 시나리오

### 월요일: 후보 수집

- 일본 후보 5개, 대만 후보 5개를 수집한다.
- Qoo10 랭킹·검색, 현지 경쟁 셀러, Rakuten·momo·Shopee 등에서 확인한다.
- 한국에서는 올리브영과 다이소를 확인해 공급 후보와 가격대를 비교한다.
- 후보마다 출처 URL, 수집일, 국가, 카테고리, 제품명을 입력한다.

### 화요일: 키워드와 수요 확인

- 제품명, 문제, 효과, 사용 장면, 구매 의도 키워드를 수집한다.
- Google Trends에서 목표 국가별 12개월 추세를 확인한다.
- Qoo10·YouTube·TikTok 자동완성에서 현지 표현을 확인한다.
- 상승 검색어는 참고하되 판매 증거로 단독 사용하지 않는다.

### 수요일: 교차검증

- 목표 국가 판매·리뷰 근거를 확인한다.
- 올리브영·다이소·공식 공급처의 조달 가능성을 확인한다.
- 경쟁 상품의 가격, 용량, 구성, 배송비를 비교한다.
- TikTok과 YouTube에서 실제로 보여줄 수 있는 사용 장면을 확인한다.

### 목요일: 점수와 의사결정

- `상품평가기준표.md`의 현재 버전으로 모든 후보를 평가한다.
- 상위 후보 3개를 `추천`, 근거가 부족한 후보를 `보류`, 위험한 후보를 `탈락`으로 분류한다.
- 점수가 높은 이유와 낮은 이유를 한 문장씩 기록한다.

### 금요일: 콘텐츠 패키지 작성

추천 후보에 대해 다음을 작성한다.

- 핵심 키워드 3~5개
- 타깃 고객
- 한 문장 포지셔닝
- 경쟁 상품 대비 차이
- 상세페이지 섹션 구성
- 필요한 실제 사진 목록
- AI 이미지가 가능한 보조 장면
- 현지 언어 검수 목록

## 2. 상품 선정 흐름

```text
후보 등록
  ↓
목표 국가 수요 확인
  ↓
한국 공급·다이소 대체품 확인
  ↓
검색·소셜·경쟁 가격 교차검증
  ↓
고정 점수표 평가
  ↓
추천 / 보류 / 탈락 / 추적
  ↓
상위 후보 상세페이지·콘텐츠 제작
  ↓
다음 주 성과와 비교
```

## 3. 목표 국가별 확인 순서

### 일본

1. Qoo10 Japan 뷰티 랭킹과 검색 결과
2. Rakuten 뷰티 주간 랭킹
3. @cosme의 리뷰·수상·반복 노출
4. 일본어 YouTube·TikTok 검색
5. 일본 경쟁 셀러의 가격·구성·배송 조건

### 대만

1. Qoo10 Taiwan 검색·랭킹
2. momo·Shopee Taiwan·POYA의 가격과 구성
3. 번체 중국어 검색어와 리뷰 표현
4. 대만 YouTube·TikTok 콘텐츠
5. 현지 셀러의 배송·정품·번체 표기 방식

올리브영은 두 국가의 실제 수요를 대신하는 출처가 아니라, 한국에서 어떤 상품을 공급할 수 있는지 확인하는 출처로 사용한다.

## 4. 키워드 수집 체계

상품마다 아래 4종류의 키워드를 기록한다.

| 종류 | 예시 | 사용처 |
|---|---|---|
| 상품명 | 리들샷, 토너패드 | 검색·상품명 |
| 문제 | 모공, 진정, 건조 | 상세페이지 제목 |
| 사용 장면 | 밤 루틴, 여행용 | 영상·이미지 |
| 구매 의도 | 추천, 비교, 리뷰, best | 검색 광고·콘텐츠 |

한국어 키워드를 그대로 번역하지 않고, 목표 국가 플랫폼의 자동완성·리뷰·댓글에서 실제 표현을 확인한다.

## 5. 상세페이지 기본 구성

1. 첫 화면: 누구에게 필요한 상품인지
2. 핵심 기능: 한 문장으로 이해되는 차별점
3. 사용 장면: 언제, 어떻게 사용하는지
4. 성분·용량·구성: 사실 정보 중심
5. 경쟁 비교: 가격·용량·기능의 차이
6. 사용법: 단계별 이미지 또는 짧은 영상
7. 주의사항: 피부 자극, 사용 주기, 보관법
8. 배송·정품·교환: 구매 불안을 줄이는 정보
9. 현지 언어 검수: 검색어와 고객 표현 반영

실제 제품 사진과 공식 자료를 우선 사용한다. AI 이미지는 배경·분위기·사용 장면 보조용으로 사용하고, 제품 포장·라벨·성분·용량·효능 전후 이미지를 임의로 바꾸지 않는다.

## 6. 자동화 전환 조건

다음 조건을 충족하기 전에는 완전 자동 추천으로 전환하지 않는다.

- 최소 8주 이상의 주간 스냅샷 축적
- 후보별 출처 URL과 점수 기록 유지
- 선정 상품과 실제 판매 결과 비교
- 점수와 결과가 반복적으로 어긋나는 항목 확인
- 규제·공급 확인 절차가 별도로 작동

그 이후 자동화는 다음 순서로 진행한다.

1. 링크와 후보 자동 수집
2. 상품명·가격·순위 정규화
3. 키워드와 중복 상품 정리
4. 점수 계산 보조
5. 사람이 근거 승인
6. 추천 후보 확정

자동화가 추천하더라도 최종 승인자는 사용자로 유지한다.

## 7. 매주 남겨야 하는 기록

- 기준 버전
- 조사한 국가와 플랫폼
- 후보 목록
- 각 후보의 근거 URL
- 항목별 점수
- 선정·보류·탈락 사유
- 다음 주 확인할 내용
- 실제 판매·클릭·문의 결과

이 기록이 쌓이면 “왜 이 상품을 골랐는가”를 다시 확인할 수 있고, 운영 숙련도에 따라 가중치 프리셋을 안전하게 바꿀 수 있다.
