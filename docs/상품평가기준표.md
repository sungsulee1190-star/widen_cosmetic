# K-Beauty 상품 평가 기준표

작성일: 2026-07-22
기준 버전: `scorecard-v1`
적용 원칙: 한 번 정한 기준은 해당 버전의 운영 기간 동안 임의로 변경하지 않는다.

## 1. 평가 원칙

- 모든 후보는 동일한 표로 평가한다.
- 점수만 입력하지 말고 근거 URL과 확인일을 함께 기록한다.
- 출처 하나만으로 최고 점수를 주지 않는다.
- 근거가 없으면 추정하지 않고 `미확인`으로 표시한다.
- 점수표를 바꿀 때는 새 버전을 만든다. 과거 점수는 다시 계산하지 않는다.

## 2. 기본 점수표

각 항목은 0~5점으로 평가하고, 가중치를 곱해 100점으로 환산한다.

| 영역 | 평가 항목 | 가중치 | 0점 기준 | 5점 기준 |
|---|---|---:|---|---|
| 목표 국가 수요 | 일본·대만 현지 랭킹/판매 신호 | 20 | 확인 근거 없음 | 주요 플랫폼에서 반복 확인 |
| 목표 국가 수요 | 리뷰·판매 지속성 | 10 | 리뷰와 이력 부족 | 최근 및 과거 기록이 모두 안정적 |
| 검색 수요 | Google Trends·플랫폼 검색 추세 | 10 | 하락 또는 검색 부족 | 상승 또는 12개월 안정 |
| 공급 가능성 | 한국에서 합법적·안정적 조달 | 10 | 공급처 불명확 | 공급처·가격·재고 확인 |
| 가격 경쟁력 | 목표 국가 판매가와 원가 여유 | 10 | 가격 경쟁 어려움 | 배송·수수료 후에도 여유 있음 |
| 소셜 가능성 | 영상·사용 장면·차별점 | 10 | 보여줄 장면이 없음 | 짧은 영상 소재가 여러 개 |
| 상세페이지 난이도 | 설명·번역·이미지 제작 | 10 | 설명과 이미지가 어려움 | 구조가 명확하고 제작 쉬움 |
| 재구매 가능성 | 반복 구매·비계절성 | 10 | 일회성·강한 계절성 | 일상 사용·반복 구매 가능 |
| 운영 위험 | 규제·배송·클레임 위험 | 10 | 위험이 높거나 미확인 | 위험이 낮고 확인 완료 |

## 3. 점수 해석

- `80~100점`: 우선 검토. 공급과 규제 확인 후 상세페이지 제작
- `65~79점`: 후보 유지. 부족한 근거를 보완
- `50~64점`: 보류. 다음 주 재검증
- `0~49점`: 탈락 또는 장기 보관

단, 아래 조건은 총점과 관계없이 즉시 보류한다.

- 정품·공급 경로를 확인할 수 없음
- 목표 국가의 화장품 표시·수입·광고 기준을 검토하지 않음
- 과장된 의학적 효능이나 전후 효과를 사용해야만 판매 가능함
- 판매가와 배송비를 포함한 가격 경쟁력이 없음

## 4. 출처별 역할

| 출처 | 주된 역할 | 점수에 반영하는 방식 |
|---|---|---|
| Qoo10 Japan/Taiwan | 목표 국가의 판매·가격·리뷰 | 목표 국가 수요 점수의 핵심 근거 |
| Rakuten Japan | 일본 카테고리 판매 흐름 | 일본 수요의 보조 근거 |
| Olive Young | 한국 내 인기와 공급 후보 | 공급 가능성·한국 수요 참고 |
| 다이소 | 저가 대체품과 접근성 | 가격 경쟁력·차별화 확인 |
| Google Trends | 검색 관심도와 지속성 | 검색 수요 점수 |
| TikTok Creative Center | 해시태그·콘텐츠·지역 트렌드 | 소셜 가능성 점수 |
| YouTube | 실제 사용법·리뷰·댓글 | 소셜 가능성·고객 언어 확인 |
| 현지 경쟁 셀러 | 현지 가격·구성·표현 | 가격·상세페이지 비교 |

## 5. 교차검증 통과 조건

추천 상태로 올리려면 아래 4개 중 최소 3개가 충족되어야 한다.

1. 목표 국가 판매 플랫폼 근거
2. 검색 추세 또는 리뷰 지속성 근거
3. 안정적인 공급·가격 근거
4. 콘텐츠와 상세페이지 제작 가능성 근거

단기 소셜 화제성만 있고 판매·공급 근거가 없으면 `추천`이 아니라 `관찰` 상태로 둔다.

## 7. 가중치 프리셋

초기에는 `초보자형`을 기본값으로 사용한다.

| 프리셋 | 강조하는 요소 | 사용 시점 |
|---|---|---|
| 초보자형 | 공급 안정성·가격·위험 | 처음 운영할 때 |
| 균형형 | 수요·콘텐츠·공급 균형 | 기본 운영이 익숙해졌을 때 |
| 공격형 | 검색 상승·소셜 확산 | 실험 상품을 찾을 때 |
| 일본형 | Qoo10 Japan·Rakuten·일본어 검색 | 일본 집중 주간 |
| 대만형 | Qoo10 Taiwan·momo·중문 검색 | 대만 집중 주간 |

프리셋별 변경은 새 `scorecard-v2`처럼 버전으로 저장하며, 특정 상품에 유리하도록 주중에 기준을 바꾸지 않는다.
