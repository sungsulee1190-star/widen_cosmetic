## 3. 첫 샘플 SKU 선정 프레임워크

### 3.1 SKU 선정 스코어카드 (8항목 × 가중치 = 100점 만점)

앞으로 **모든 신규 SKU를 이 스코어카드로 평가**한다.

| # | 평가 항목 | 가중치 | 1점 (최저) | 3점 (보통) | 5점 (최고) |
|---|----------|--------|-----------|-----------|-----------|
| A | **해외 수요 확인** | ×4 (20%) | 검색량 미미, SNS 언급 없음 | Qoo10 검색 시 일부 노출 | Qoo10 랭킹 상위 + @cosme/LIPS 버즈 + SNS 바이럴 |
| B | **국내 소싱 용이성** | ×4 (20%) | 브랜드 직거래만 가능 | 올리브영 온라인 구매 가능 | 다이소/올리브영/무신사 상시 재고 + 도매 가능 |
| C | **예상 마진** | ×3 (15%) | 순마진 5% 미만 | 순마진 5~10% | 순마진 10%+ 또는 인디 브랜드 20%+ |
| D | **경쟁 셀러 포화도** (역산) | ×3 (15%) | 100+셀러 포화 | 30~50셀러 | 10셀러 미만 블루오션 |
| E | **성분/트렌드 지속성** | ×2 (10%) | 1개월 유행 | 3개월 노출 | 6개월+ 상승 또는 스테디셀러 |
| F | **Fast Copycat 가능성** | ×2 (10%) | 벤치마킹 페이지 없음 | 경쟁 페이지 1~2개 | 잘 만든 판매 페이지 5개+ |
| G | **규제 리스크** (역산) | ×1 (5%) | 금지 성분/의약부외품 분류 | 일부 표현 제한 | 리스크 없음 |
| H | **카테고리 다양성** | ×1 (5%) | 이미 같은 카테고리 3개+ | 카테고리 1개 겹침 | 포트폴리오에 없는 신규 카테고리 |

**총점 = (A×4 + B×4 + C×3 + D×3 + E×2 + F×2 + G×1 + H×1)**

| 등급 | 점수 | 판단 |
|------|------|------|
| **S** | 85~100 | 즉시 테스트 등록 |
| **A** | 70~84 | 우선 조사 대상 |
| **B** | 55~69 | 추가 조사 후 결정 |
| **C** | 40~54 | 보류 |
| **D** | 39 이하 | 제외 |

- **목적:** 오프라인 마케팅 키워드·언어 캡처
- **꼭 담을 것:** 효능 키워드, 성분 강조, 다국어 표현

#### STEP 4: 재고 상태 샷
- **목적:** 수요 수준 판단 (품절 = 해외 기회 신호)
- **꼭 담을 것:** 빈 자리, "입고 예정" 표시, 잔여 수량

### 6.3 촬영 시 유의사항

- **고객 얼굴 노출 금지** — 각도 조절
- **촬영 제한 확인** — "개인 리서치 용도"임을 밝히기
- **오전 10시 직후 / 저녁 8시 이후** 방문 시 촬영 수월
- **사진마다 즉시 메모** — 카카오톡 나에게 보내기: "SKU명, 매장, 가격, 한 줄 소감"

## 7. MD 리포트 & 사진 파일명 규칙 (정밀 설계)

### 7.1 설계 원칙 4가지

1. **날짜순 자동 정렬** — 탐색기에서 이름순 = 시간순
2. **즉시 검색** — 아무 키워드 2개 조합이면 찾을 수 있음
3. **파일명만 보고 내용 추론** — 폴더를 열지 않아도 내용 파악
4. **입력 부담 최소화** — 약어 10개 미만

### 7.2 약어 사전

#### 채널 약어 (2~3자)

| 약어 | 의미 |
|------|------|
| `Q10` | Qoo10 JP |
| `SPE` | Shopee TW |
| `OY` | 올리브영 |
| `DS` | 다이소 |
| `MSB` | 무신사뷰티 |
| `SNS` | TikTok/Instagram/YouTube |

#### 목적 약어 (3~5자)

| 약어 | 의미 | 예시 상황 |
|------|------|----------|
| `COPY` | Fast Copycat 분석 | 경쟁 판매 페이지 해부 |
| `PRICE` | 가격 비교/모니터링 | 국내외 가격 비교 |
| `VISIT` | 오프라인 매장 방문 | 명동/성수/홍대 |
| `PHOTO` | 매대/제품 사진 | 진열 상태 캡처 |
| `REVIEW` | 리뷰 분석 | 고객 리뷰 수집 |
| `TREND` | 트렌드 관찰 | 인기 급상승 분석 |

#### 브랜드 약어 (긴 이름만)
