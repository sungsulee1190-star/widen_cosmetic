# WIDEN K-Beauty 브랜드 인텔리전스 대시보드 설계서 (Design Spec)

작성일: 2026-07-04  
프로젝트: `widen_cosmetic` (위든 K-Beauty 역직구 대시보드)  
작성자: AI 파트너 (Antigravity)  
호스팅 대상: GitHub Pages (정적 웹 호스팅)  
데이터 백엔드: Google Sheets (구글 스프레드시트 웹 게시 연동)

---

## 1. 아키텍처 및 시스템 개요

본 시스템은 1인 운영자인 피터님이 대만(Shopee TW) 및 일본(Qoo10 JP) K-Beauty 시장 동향을 파악하고 주간 소싱/운영 의사결정을 내릴 수 있도록 돕는 **단일 파일 모던 SPA (Single Page Application)** 웹 대시보드입니다.

```
+-------------------------------------------------------------+
|                     [ WIDEN Web Dashboard ]                 |
|  1. HTML5 / CSS3 (Vanilla CSS Custom Theme, Glassmorphism)   |
|  2. Vanilla JavaScript (State Management, Tab Switcher)     |
|  3. LocalStorage (Decision History Sync)                    |
+-------------------------------------------------------------+
                              |
                     (Fetch CSV / Fallback)
                              v
+-------------------------------------------------------------+
|                  [ Google Sheets DB (Web) ]                 |
|  - Brands Sheet: Brand Info & SWOT                          |
|  - SKU Sheet: Time-Series SKU stats (Price, Rank, Margins)  |
|  - Trends Sheet: Cumulative Trend Signals                   |
+-------------------------------------------------------------+
```

### 핵심 기술 스택
- **구조 & 디자인**: HTML5, Vanilla CSS3 (에스테틱 뷰티 감성의 보라/로즈 그라데이션, Glassmorphism 테마)
- **로직**: Vanilla JavaScript (ES6+)
- **데이터 저장 및 관리**:
  - **시계열/누적 데이터**: Google Sheets (Publish to Web 기능을 이용해 CSV 링크 연동)
  - **사용자 액션 상태**: 브라우저 `localStorage` (새로고침해도 의사결정 상태 보존)
- **배포**: GitHub Pages (`https://github.com/sungsulee1190-star/widen_cosmetic` 저장소 배포)

## 2. 화면 구조 및 UI 디자인 (Aesthetics)

### 시각 디자인 컨셉
- **Theme**: Premium Aesthetic Beauty & Sleek Dark-Soft Contrast
- **Color Palette**:
  - `--bg-main`: `#f8fafc` (연한 슬레이트 블루-그레이 백그라운드)
  - `--surface-card`: `rgba(255, 255, 255, 0.85)` (반투명 백색 글래스모피즘 카드)
  - `--border-color`: `#e2e8f0` (부드러운 테두리 경계선)
  - `--accent-gradient`: `linear-gradient(135deg, #8b5cf6 0%, #ec4899 100%)` (로얄 바이올렛에서 로즈 핑크로 이어지는 고급 뷰티 그라데이션)
  - `--text-primary`: `#0f172a` (선명하고 짙은 슬레이트 텍스트)
  - `--text-secondary`: `#64748b` (설명용 보조 회색 텍스트)
- **Typography**: `Inter` (숫자 및 영문), `Noto Sans KR` (국문 가독성 극대화)

### 레이아웃 구성
좌측 사이드바와 우측 메인 영역의 2분할 구조로 모바일 및 데스크톱 반응형 레이아웃을 구현합니다.

#### [좌측 사이드바 (Navigation Sidebar)]
- **WIDEN 로고 및 슬로건**
- **대메뉴 3개**:
  1. `📊 메인 대시보드` (기본 선택)
  2. `📈 트렌드 아카이브`
  3. `🌿 인디브랜드 도감`
- **하단 메타 바**: 최신 업데이트 일자, 구글 시트 연동 상태(연동됨/로컬데이터 사용) 표시

#### [우측 메인 영역 (Main Panel)]
사이드바 메뉴 클릭 시 부드러운 페이드인 애니메이션과 함께 해당 화면으로 전환됩니다.

## 3. 상세 화면 설계

### 3.1 메인 대시보드 (3개 탭 전환 구조)

#### 탭 1: 주간 의사결정 (Weekly Decisions)
- **오늘의 의사결정 카드**: 피터님이 즉각 판단해야 할 긴급 액션 아이템 리스트.
  - 의사결정 항목별로 `채택`, `보류`, `추가 조사` 버튼 제공.
  - 버튼 클릭 시 상태가 즉시 변경되며, `localStorage`에 자동 저장되어 영구 보존.
  - 결정 근거와 예상 효과를 비주얼 뱃지로 노출.
- **이번 주 업데이트 상품 (Top 5)**: Shopee/Qoo10 등록 상품 중 성과 이상 신호가 있는 제품 수정 제안.
  - 수정 항목: 상품명 수정, 썸네일 교체, 가격 조정, 상세페이지 보강 등.
- **신규 소싱 후보 (Top 5)**: 이번 주 소싱 가치가 높은 신제품 목록.
- **유입/전환 이상 신호**: 내 채널의 노출 대비 클릭률(CTR), 클릭 대비 전환율(CVR) 이상 신호 경고 카드.

#### 탭 2: 브랜드 & SKU 인텔리전스 (3-Layer 구조)
- **Layer 1 (Market Filter)**: 상단에 대만(Shopee TW) / 일본(Qoo10 JP) 선택 버튼 제공.
- **Layer 2 (Brand Grid)**: 선택한 국가에서 활성화된 5대 핵심 브랜드 그리드 노출.
  - 브랜드 카드 내에 주간 변동 모멘텀(평균 순위 변동, 주간 리뷰 증가 합계)을 요약.
- **Layer 3 (SKU Accordion)**: 브랜드 카드 클릭 시 하단에 펼쳐지는 5개 제품 레벨 아코디언.
  - 5개 브랜드 x 5개 제품 = 총 25개 SKU의 상세 현황.
  - 출력 정보: 현지가(TWD/JPY), 현재 순위, 주간 리뷰 증가량, 마진율 계산 수식 시각화.
  - **마진율 계산 수식**: `판매가 - 원가 - 플랫폼 수수료 - 해외 배송비 = 순수익 (마진율 %)`을 한눈에 알기 쉽게 도식화.

#### 탭 3: 트렌드 & 경쟁사 요약 (Summary)
- **이번 주 트렌드 요약**: 급상승 중인 카테고리와 대표 브랜드 요약.
- **경쟁사 변화 요약**: 경쟁사 3~4곳의 신상품 등록 현황, 가격 변동 캡처 이력.

---

### 3.2 트렌드 아카이브 (Trend Archive)
- **검색 및 필터**: 카테고리(선케어, 패드, 앰플 등) 및 국가(대만/일본)별 필터 칩.
- **타임라인 목록**: 과거부터 축적되어 온 트렌드 시그널 데이터를 최신순 정렬 리스트로 노출.
- **데이터 누적 뷰**: 각 주차별 트렌드 동향을 세로 타임라인 레이아웃으로 직관적 확인 가능.

---

### 3.3 인디브랜드 도감 (Indie Brand Archive)
- **"잘나가는 브랜드" 필터링 대시보드 (기준 탑재)**:
  - `🔥 급성장`: 주간 리뷰 증가율 15% 이상 또는 순위 급상승
  - `💰 고마진`: 마진율 30% 이상 확보 가능한 브랜드
  - `📈 소셜 대세`: SNS 언급량 및 한국 올리브영 랭킹 상승세
  - 필터 버튼 클릭 시 해당 조건에 부합하는 브랜드만 그리드에 동적 필터링 처리.
- **브랜드 상세 정보 카드**:
  - 브랜드명, 브랜드 히스토리, 연 매출 규모 추정.
  - 강점 섹터 및 대표 제품, 가격대 수준.
  - **SWOT 분석 매트릭스**: 2x2 그리드로 강점(S), 약점(W), 기회(O), 위협(T)을 뷰티 전문 비즈니스 관점에서 보기 좋게 박스 디자인으로 시각화.

## 5. 검증 계획 (Verification Plan)

1. **로컬 실행 검증**: HTML 파일을 직접 크롬/엣지 브라우저에서 열어 사이드바 내비게이션, 탭 전환, 아코디언 열기/닫기가 정상 작동하는지 확인합니다.
2. **상태 보존 검증**: 의사결정 탭에서 특정 아이템을 '채택'으로 설정한 후 새로고침했을 때 동일 상태가 유지되는지 확인합니다.
3. **구글 시트 연동 Fallback 테스트**: 구글 시트 URL이 없거나 연결에 실패할 때, 내장된 실제 데이터를 바탕으로 대시보드가 에러 없이 정상적으로 렌더링되는지 확인합니다.
4. **브랜드 필터 작동 검증**: '고마진', '급성장' 필터를 각각 눌렀을 때 조건에 부합하는 브랜드 카드만 필터링되어 화면에 출력되는지 확인합니다.
